# project-seabling
Seabling mon is a difficultly and exploration packed based in Cobblemon
This project is meant to be used in the Seabling Modpack, currently in development.
Currently the pack is developed using kubejs as a base to allow gloabl loading of the datapack features, but may expand into koltin code.

## Planed Features
### Difficutly options for gym leaders & key trainers
- Two difficulties
    - Competitive: designed to be test the up limits of even the most dedicated trainers, with teams that are viable in the          competive pokemon scene. This is the packs authour intended experience.
        - Gym party caps: when facing a gym leader, you cannot have more team memebers than the gym leader.
        - To respect the title of champion, the teams will be drawn from the irl pokemon word championship master divsion                champion teams.    
    - Mainline: designed to be onpar with the the more tradtional challenge curve of the Pokemon series.
### Gym Leaders
- 18 custom gym leaders with 4 different teams for different badge stages.
    - To advance to the elite 4, only 8 trainers need to be defated, but all can be challenged.
- 4 custom elite 4 memebers
- 1 champion
- A range of npcs, trainers, and special trainers tied to the tale of the pack.
